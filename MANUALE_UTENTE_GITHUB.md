# Manuale Utente - Invalsi Scheduler Pro

Benvenuto in **Invalsi Scheduler Pro**! Questo software è stato creato per automatizzare la creazione del calendario delle prove Invalsi nel tuo Istituto Scolastico.

## 1. Preparazione dei Dati
Il software necessita del file orario della scuola in formato **XML**. Questo file deve essere generato tramite l'esportazione dati del software **"Orario Facile"**.

1. Apri Orario Facile.
2. Vai su *File -> Esporta -> XML*.
3. Salva il file sul tuo computer.

## 2. Caricamento nell'applicazione
1. Avvia `InvalsiScheduler.exe`.
2. Nella prima scheda **"Dati & XML"**, clicca su **"Carica File XML Orario"**.
3. Seleziona il file salvato al punto precedente. Il sistema analizzerà automaticamente classi, aule e materie.

## 3. Configurazione
Passa alla scheda **"Configurazione"**:
- **Generale**: Imposta le date di inizio e fine delle prove e seleziona i giorni della settimana in cui si possono svolgere.
- **Classi**: Seleziona le classi che devono sostenere le prove.
- **Materie**: Configura la durata in ore per ogni prova. Puoi aggiungere nuove materie o rimuovere quelle predefinite.
- **Aule**: Seleziona le aule informatiche/laboratori abilitati alle prove.

**Opzione "Priorità Ore":** Se attivata, il sistema cercherà di incastrare prima le materie che durano più ore, rendendo più probabile il completamento dello scheduling senza conflitti.

## 4. Generazione
Nella scheda **"Esporta"**:
1. Clicca su **"Genera Scheduler"**.
2. Verrà mostrata la lista di tutte le prove pianificate.
3. Se necessario, clicca su **"Modifica"** accanto a una prova per spostarla manualmente o far cercare uno slot alternativo al sistema.

## 5. Salvataggio Risultati
Clicca su **"Scarica CSV/XLS"** per ottenere:
- Un file **CSV** (ottimo per importazioni in altri sistemi).
- Un file **Excel (.xlsx)** formattato con fogli separati (ordine cronologico, per classe, per aula).
